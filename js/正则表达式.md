### 任意字符（除换行符）
```
.
```

### 字母或数字或下划线或汉字
```
\w
```

### 任意的空白符
```
\s
```

### 数字
```
\d
```

### 单词的开始或者结束
```
\b
```

### 匹配开始
```
^
```

### 匹配结束
```
$
```

### 匹配次数
```
// >=0
*
// >=1
+
// 0 或者 1
?
// n次
{n}
// >=n
{n,}
// >=n  <=m
{n,m}
```

### 指定字符集或范围，[]
```
[abcdefg]
[0-9]
[a-z]
```

### 分支 |
```
a|b
```

### 分组，用小括号括起来，()
```
(abc){3}
```
